Chapter 3 Operating Systems

Terminology

Term Definition
Operating System A program that controls the excecution of application programs and acts as am interface between the user of a computer and the computer hardware.
Job Queue This queue keeps all the processes in the system.
FIFO First in, First out. A method for handling data structures where the first element is processed first and the newest element is processed last.
Interactive Processing In a system, processing in which a computer program or a procedure directs the system to alternately accept input and respond to input.
Time-Sharing The sharing of a computing resource among many users at the same time by means of multiprogramming and multi-tasking.
Multiprogramming The technique of utilizing several programs concurrently in a single computer system.
Application Software Software designed to handle specific tasks for users.
System Software A type of computer program that is designed to run a computer's hardware and application programs.
Utility Software System software designed to help analyze, configure, optimize or maintain a computer. Used to support the computer infrastructure.
Graphical User Interface (GUI) A computer program that enables a person to communicate with a computer through the use of symbols, visual metaphors, and pointing devices.
Shell A software program that interperates commands from the user so that the operating system can understand them and perform the appropriate functions.
Kernel A computer program at the core of a computer's operating system and generally complete control over everything in the system.
Device Driver A computer program that acts as an intermediary between the operating system and a device such as a disk drive, video card, printer or keyboard.
File Manager System software responsible for the creation, deletion, modification of the files and managing their access, security and the resources used by them.
Memory-Manager The memory-manager of an operating system handles the main memory of a computer, especailly RAM.
Virtual Memory A memory management technique that provides an "idealized abstraction od the storage resources that are actually avalible on a given machine.
Boot Strapping A process that automatically loads and executes commands.
ROM (Read-Only Memory) Memory that can only be read from and is not electronically modifiable after being manufactured. Used to store critical foundational information like startup processes and software instructions.
Deadlock A condition where a program cannot access a resource it needs to continue.
Spooling A specialized form of multi-programming for the purpose of copying data between different devices.
Super User (or administrator) A special user account for general system administraction such as networks and databases.
Multi-Factor Authentifacation Requires a user to present evidence from multiple distinct factors.
