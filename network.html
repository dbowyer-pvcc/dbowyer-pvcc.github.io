Chapter 4: Networking & the Internet

Topic Definitions
Network Classifications
  • LAN: Local area network, a collection of devices connected together in one physical location, such a building, office or home.
  • MAN: Metropolitan area network. Computer network that is larger than a singe building local area that is smaller than a wide area network.
  • WAN: Wide area network. A computer communications network that spans cities, countries and the globe.
Network Models
  • Bus Network: LAN topology in which all the nodes are connected to a shared bus.
  • Star Network: LAN topology in which all nodes-PCs, workstations or other devices-are directly connected to a common central computer that is often referred to as the hub.
Protocols
  • CSMA/CD Ethernet: Carrier-sense multiple access with collision dectection. Medium access control method used most notably in early ethernet technology for loca area networking.
  • CSMA/CA WIFI: Carrier-sense multiple access. A medium access control protocol in which a node verifies the abcsence of other traffic before transmitting on a transmission medium.
Combining Networks
  • Repeater: Electronic device that relays a transmitted signal.
  • Bridge: Used to connect two subnetworks that use interchangeable protocols.
  • Switch: A small hardware device that links multiple PCs and devices together within one local area network, wide area network and different network topology.
  • Router: Recieves and sends data on computer networks.
  • Gateway: A hardware device that acts as a "gate" between two networks.
Client/server Model
  • P2P: Peer-to-peer. Computer networking using a distributed archetecture.
  • Distributed System: A system whos components are located on different networked computers, which communicate and coordinate their actions by passing messeges to one another.
  • Cloud Computing: Enables remote access, primarily through the internet, to shared resources; typically being maintained by a third-party.
Internet Addressing
  • Tier 1: IP network that can reach every other network on the internet soley via settlement-free interconnection.
  • Tier 2: An internet service provider which engages in the practice of peering with other networks, but which also purchases IP transit to reach some portion of the internet.
  • Tier 3: Primarily engaged in delivering internet access to end customers.
IP Address
  • ICANN: Internet Corporation of assigned names and numbers. Nonprofit corporation with responsability for IP adress space allocation, protocol parameter assignment, domain name system management and root server system management functions.
  • DNS: Domain name system. Turns domain names into IP addresses, which browsers use to load internet pages.
  • TLD: Top-level domain. The last segemnt of a domain name, or the part that follows immediately after the "dot" symbol.
  • Country-code TLDs: A TLD name that is used to define the domain for a particular counrty or a geographical area.
Internet Protocols
  • FTP: File transfer protocol, a standard communication protocol used for the transfer of computer files from a server to a client on a computer network.
  • HTTP: Hypertext transfer protocol, is standard application-level protocol used for exchanging files on the world wide web.
  • SMTP: Simple mail transfer protocol is a technical standard for transmitting email over a network.
World Wide Web
  • URL: Unifrom resource locator. The address of a given unique resource on the web.
  • Client-Side Activities: Everything in a web application that is displayed or takes place on the client "end user" device.
  • Server-Side Activities: Everything that happens on the server.
Internet Four-Layer Software
  • Application Layer: Layer that users interact with and use.
  • Transport Layer: Responsible for delivering data to the appropriate application process on the host computers.
  • Network Layer: The part of the internet communications process occour, by sending packets of data back and forth between different networks.
  • Link Layer: The protocol layer in a program that handles the moving of data into and out of a physical link in a network.
Internet Is:
  • Packet-Switching Network: It enables the sending of data or network packets between a source and destination node ocver a network channel that is shared between multiple users and/or applications.
  • Fault Tolerant: Capability of a computer system, electronic system or network to deliver uninterrupted service.
  • Redundant: "Expecting what is normal". Duplicate devices that are used for backup purposes.
Cybersecurity
  • Malware: Term that describes any malicious program or code that is harmful to systems.
  • Virus: Malicious computer program. When executed, replicates itself and inserts its own code.
  • Worm: Malicious, self-replicating software program which affects the functions of software and hardware programs.
  • Trojan Horse: Malicious software that misleads users of its true purpose.
  • Spyware: Software with malicious behavior that aims to gather information about a person or organization and send such information to another entity in a way that harms the user.
  • Phishing: Cybercrime in which a target is contacted by email, telephone or text messege by someone posing as a legitmate institution to lure individuals into providing sensitive data.
  • DDos Attack: Distributed denial-of-service attack, malicious attempt to disrupt the normal traffic of a targeted server, servic or network by overwhelming the target or its surrounding infratructure with a flood of internet traffic.
Protection
  • Firewall: Network security device, either hardware or software-based, which monitors all incoming and outgoing traffic and based on a defined set of security rules it accepts, rejects or drops that specific traffic.
  • Spam Filter: Sorting solution that helps users in detecting and weeding out spam.
  • Proxy Server: Acts on the behalf of the user.
  • Antivirus Software: A program that is created to search, detect, prevent and remove software viruses from a system that can be harmful.
  • Cryptography: Secure information and communication techniques.
  • Public Key Encryption: Asymmetric encryption technique which uses different keys for enceyption and decryption, allowing computers over the internet to securly communicate with each other.
