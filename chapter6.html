Chapter 6: Programming Languages

Term Definition
Machine Language The set of symbolic instruction codes, usually in binary form, that is used to represent operations and data in a machine.
Debugging The process of finding and fixing errors or bugs in the source code of any software.
Assembler A program that converts a program written in assimbly language into machine code.
Assembly Language Type of low-level computer programming language consisting mostly of symbolic equivalents of a particular computer's machine language.
Translator A programming language processor that connets a computer program from one language to another.
Compiler A computer program that translates computer code written in one programming language into another language.
Interpertor A computer program that directly executes instructions written in a programming or scripting language, without requiring them previously to have been compiled into a machine language program.
Cross-Platform Software Software that is designed to work in several computing plaforms.
Natural Language An ontology-assisted way of programming in terms of natural language sentances, e. g. english.
Formal Language Consists of words whos letters are taken from an alphabet and are well-formed according to a specific set of rules.
Machine Independence Refers to something that is not tied or dependent on a specific type of computer or computer hardware. Can be used to describe software, programming languages, data formats, or other elements of computer systems.
High-Level Programming Language A programming language that resembles natural language or mathematical notation.
Object-Oriented Programming (OOP) A programming paradigm based on the concept of "objects", which contain data and code.
Scripting Language A programming language that is used to manipulate, customize, and anotate the facilities of an existing sytem.
Data Types
  • Intiger: A data type that represents some range of mathematical intigers.
  • Float: Data type composed of a number that is not an intiger, because it includes a fraction represented in decimal format.
  • String: A data type often implimented as an array data structure of bytes that stores a sequence of elements, typivslly charicters, using some charicter encoding.
Primative Data Type A set of basic data types from which all other data types are constructed.
Data Structure Data organization, management, and storage formulas that are chosen for efficent access to data.
Python Collection Data Structures
  • List: One of the fundamental data structures in Python when building a project in Pyhon is a list, which is defined as an ordered collection of items.
  • Tuple: A built-in data structure in Python that reperesents an ordered collection of things, tuples provide less funtionality than lists do.
  • Set: Consist of a group of distinct, unordered elements. This means that even if the data is repeated multiple times, it will only be entered into the set once.
  • Dictionary: An ordered collection of data values, used to store data values like a map.
Function A named section of a program that performs a specific task.
Local Variable A variable that is given a local scope.
Global Variable A variable with global scope, meaning that it is visible throughout the program, unless shadowed.
Source Program An origonal computer program written by a programmer that is converted into the equivalent object program, written in a machine language, by the compiler or assimbler.
Object Program A variable, a data structure, a function, or a method.
